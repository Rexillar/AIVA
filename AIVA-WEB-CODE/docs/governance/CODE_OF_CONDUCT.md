# Code of Conduct (Governance)

All contributors, maintainers and community members are expected to follow this Code of Conduct.

1. Be respectful: Assume good intentions. Harassment or hateful language will not be tolerated.
2. Be constructive: Discuss facts, not people. Offer suggestions instead of criticism alone.
3. Respect privacy: Do not share private information without explicit permission.
4. Report issues: If you encounter harassment, please follow `SECURITY.md` to report privately.

Violation of the Code of Conduct may result in warnings, temporary suspension, or permanent ban from the community.
