## 📈 What Gets Built

### Backend (3 files, 930+ lines)
```javascript
// Source Model
Source {
  title, authors, doi, url,
  types: [academic_paper, website, article, interview],
  citations: { apa, mla, chicago, harvard },  // AUTO-GENERATED
  workspace, addedBy, createdAt
}

// Knowledge Index Service
buildKnowledgeIndex(workspaceId) → {
  summary: { totalNotes, totalSources, healthScore },
  topContributors: [...],
  topTopics: [...],
  recommendations: [...]
}

// Source API
POST   /api/sources                          → Create
GET    /api/sources?workspace=:id            → List
GET    /api/sources/:id                      → Read
PATCH  /api/sources/:id                      → Update
DELETE /api/sources/:id                      → Delete
POST   /api/sources/:id/link/note/:noteId    → Link
DELETE /api/sources/:id/link/note/:noteId    → Unlink
GET    /api/sources/:id/citation?format=apa  → Format
GET    /api/workspace/:id/knowledge-index    → Index
```

### Frontend (2 components, optional)
```jsx
<SourcePanel />

  └─ Link to notes
```
