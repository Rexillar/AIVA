# AIVA System Documentation

This is the comprehensive documentation for the AIVA AI Virtual Assistant system.

## Overview

AIVA is a deterministic execution system that provides AI-powered virtual assistance with rule-bound, state-aware, and non-emotive operations.

## Structure

This documentation is organized into the following sections:

- **Architecture**: System design and models
- **Backend**: Server-side components and services  
- **API**: API routes, controllers, and error handling
- **Frontend**: Client-side state management, pages, and components
- **Conventions**: Naming, headers, logging, and error policies
