# Naming Conventions

## Purpose

This document outlines naming conventions for code.

## File Naming

- kebab-case for files: `task-card.jsx`
- PascalCase for components: `TaskCard.jsx`
- camelCase for utilities: `formatDate.js`

## Variable Naming

- camelCase for variables and functions
- UPPER_CASE for constants
- PascalCase for classes and constructors

## Database Naming

- snake_case for table names
- camelCase for field names
