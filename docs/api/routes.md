# API Routes

## Purpose

This document lists and describes all API routes.

## Route Structure

- RESTful design
- Versioned endpoints (/api/v1/)
- Consistent response format

## Authentication Routes

- POST /api/auth/login
- POST /api/auth/register
- POST /api/auth/logout

## Task Routes

- GET /api/tasks
- POST /api/tasks
- PUT /api/tasks/:id
- DELETE /api/tasks/:id

## Other Routes

- User management
- File upload
- External integrations
